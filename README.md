# A "go, please" client

A game client made with [Ebitengine](https://ebitengine.org/) and [EbitenUI](https://github.com/ebitenui)

## Getting Started

To start the client, run:
```bash
go run cmd/game/main.go
```
## Configuration
See
```
./config/config_defaults.yaml
```
